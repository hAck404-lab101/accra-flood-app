# Live Feature Scope

Accra Flood App is a live flood-status web platform for Accra. The build includes all web-based features from the product plan except native mobile app features.

## Included Features

### 1. Basic Live Map

- Accra map
- Flood status pins
- Status categories: flooded, water rising, clear, unknown
- Latest reports list
- Area search
- Area pages

### 2. Community Reporting

- Submit flood report
- Use current location
- Add area name
- Add water level
- Add description
- Add optional reporter details
- Add photo evidence
- View submitted reports

### 3. Trust and Verification

- Admin verification
- Verified, unverified, rejected, duplicate, and stale report states
- Report expiry
- Report confidence level
- Fake report flagging
- Abuse reporting
- Community trust score

### 4. Area Intelligence

- Area risk summaries
- Known flood-prone zones
- Water level trend
- Historical flood records
- Emergency contact information
- Area management from admin

### 5. Traveller Mode

- Journey checker
- Ho to Accra route check
- Start and destination input
- Risky areas along route
- Route risk warning
- Safer route suggestions
- Before-you-travel summary

### 6. Weather Intelligence

- Rain radar overlay
- Weather alert banner
- Rainfall intensity level
- Official alert module
- Area-based risk scoring
- Time-based report freshness

### 7. Sharing and Alerts

- WhatsApp share link
- SMS alert subscription
- Web push notification-ready structure
- Public alert automation

### 8. Admin Operations

- Admin dashboard
- Admin map view
- Verification queue
- Report metrics
- Area management
- Emergency contact management
- Monitoring and analytics
- Public feedback system

### 9. Advanced Smart Features

- AI flood risk score
- Possible flood-prone area prediction
- Traffic and flood combination
- Voice report support
- Offline-friendly web mode

## Excluded Features

Native mobile app features are excluded for now:

- Android app
- iOS app
- Native live-location mobile alerts
- Native mobile background notifications
- Native mobile driver mode

The web app must still be mobile-first and feel good on phones.

## Build Priority

1. Live report submission and map display
2. Admin verification and stale reports
3. Area search and area intelligence
4. Photo evidence and report confidence
5. Traveller route checker
6. Weather/rain layers
7. WhatsApp/SMS/web alerts
8. Admin operations dashboard
9. AI and prediction features
