# Accra Flood App Roadmap

## Phase 1 — MVP

Goal: show a flood map and collect reports.

- Map of Accra
- Demo flood pins
- Supabase reports table
- Report form
- Admin verification page
- Area details page

Success metric:

> A report submitted from `/report` appears in Supabase and can be viewed on the map.

## Phase 2 — Trust and Verification

Goal: make the data safer and more trustworthy.

- Admin authentication
- Verified-only public mode
- Report abuse controls
- Report expiry, for example reports older than 6 hours become stale
- Evidence photos
- Confidence level: low, medium, high

## Phase 3 — Weather Intelligence

Goal: add weather and rainfall context.

- Rain radar overlay
- Heavy rain warning banner
- Area risk scoring
- Time-based report freshness
- Official alert feed module

## Phase 4 — Travel Mode

Goal: help people travelling from Ho or other towns into Accra.

- Start and destination input
- Route risk summary
- List of risky areas on the route
- Safer route suggestions
- WhatsApp share link

## Phase 5 — Public Launch

Goal: make it safe for public use.

- Privacy policy
- Terms and safety disclaimer
- Proper admin roles
- Domain and Vercel deployment
- Monitoring and analytics
- Public feedback system
